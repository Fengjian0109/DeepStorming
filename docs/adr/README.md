# Architecture Decision Records

ADRs record decisions that materially constrain DeepStorming. A decision is changed by adding a
new ADR that supersedes the old one, not by silently rewriting history.

| 0003 | Local-first BYOK model access            | Accepted |

| 0005 | Deterministic lesson state machine       | Accepted |
